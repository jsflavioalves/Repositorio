addSolicitacao()

addSolicitacao(string  $dh_solicitacao, \TipoTeleconsultoria  $tipo, \CanalAcesso  $canal_acesso_sincrona, string  $cpf_solicitante, string  $especialidade_solicitante, string  $ponto_telessaude_solicitacao, string  $equipe_do_solicitante, string  $tipo_profissional, array  $cids, array  $ciaps, string  $dh_resposta_solicitacao, boolean  $evitou_encaminhamento, boolean  $intensao_encaminhamento, \GrauSatisfacao  $grau_satisfacao, \ResolucaoDuvidaTeleconsultoria  $resolucao_duvida, boolean  $potencial_sof)

Adiciona a solicitação de teleconsultoria.

A teleconsultoria é identificada exclusivamente pela data da solicitação e o cpf do solicitante.

Parameters

string $dh_solicitacao

Data/hora da solicitação da teleconsultoria no formato dd/MM/yyyy HH:MM:SS
\TipoTeleconsultoria $tipo

Tipo da solicitação, se Síncrona ou Assíncrona
\CanalAcesso $canal_acesso_sincrona

Canal de acesso, se internet ou telefone
string $cpf_solicitante

CPF do profissional que solicitou a teleconsultoria
string $especialidade_solicitante

Código CBO da ocupação do solicitante no momento da solicitação da teleconsultoria
string $ponto_telessaude_solicitacao

Código CNES do estabelecimento de saúde no qual o profissional solicitante atua
string $equipe_do_solicitante

Código INE da equipe de saúde na qual o profissional solicitante faz parte
string $tipo_profissional

Código do tipo de profissional de saúde cadastrado no SMART
array $cids

Lista com os códigos CID (Classificação Internacional de Doenças).
array $ciaps

Lista com os códigos CIAP (Classificação Internacional de Assistência Primária).
string $dh_resposta_solicitacao

Data/hora da resposta da solicitação no formato dd/MM/yyyy HH:MM:SS
boolean $evitou_encaminhamento

Se a teleconsultoria evitou o encaminhamento de paciente
boolean $intensao_encaminhamento

Se o profissional registrou na teleconsultoria que tinha intenção de encaminhar o paciente
\GrauSatisfacao $grau_satisfacao

Grau de satisfação do solicitante quanto a resposta da sua teleconsultoria
\ResolucaoDuvidaTeleconsultoria $resolucao_duvida

Se a resposta da teleconsultoria atendeu, atendeu parcialmente ou não atendeu à sua teleconsultoria.
boolean $potencial_sof

Se a teleconsultoria tem pontencial para transforma em SOF
